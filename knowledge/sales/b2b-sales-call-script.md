# B2B Sales Call Script — Compact Structure

## 1. Intro (Earn right to ask questions)
- Ask why they took the call
- Clarify desired outcome
- Set agenda: goals → examples → next step
- Get agreement
- Request permission to ask questions

## 2. Discovery (PULL Framework)

### P — Project
- What are they trying to achieve?
- What metrics define success?

### U — Unavoidable
- Why now?
- Deadline or urgency?

### L — List
- What options have they considered?

### L — Limitations
- Why those options fail?

### Fallback
- Ask direct goal question
- Use examples (“slide approach”)
- If no relevance → end call early

## 3. Transition (Confirm understanding)
- Restate:
  - Goal
  - Urgency
  - Failed options
- Confirm desired solution characteristics

## 4. Supply — Concept (30–60 sec)
- Simple product description aligned to PULL
- Minimal detail
- Use visual/slide if possible
- Fit check:
  - “Does this match what you need?”

## 5. Supply — How (30–60 sec)
- Show how product achieves goal
- Address limitations
- Demo or walkthrough
- Emphasize outcome
- Fit check:
  - “Is this what you expected?”

## 6. Next Step (Advance deal)
- Ask what they need to decide
- Identify evaluation criteria
- Propose next step
- Schedule follow-up if aligned

## Core Principle
Discovery (PULL) determines the effectiveness of the entire process.
